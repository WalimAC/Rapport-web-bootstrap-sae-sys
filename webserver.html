Serveur Web

Installation Apache/Nginx et déploiement des fichiers HTML/CSS.

Installation du serveur Apache

Commandes d'installation et démarrage du service Apache :

sudo apt update
sudo apt upgrade -y
sudo apt install apache2 -y
sudo systemctl enable apache2
sudo systemctl start apache2
sudo systemctl status apache2

Déploiement HTML/CSS

Méthode de déploiement des fichiers :

  • Déploiement via WebStorm ou client FTP vers le répertoire /var/www/html/
  • Configuration des permissions : sudo chown -R www-data:www-data /var/www/html/
  • Activation du Virtual Host : sudo a2ensite sae.local
  • Redémarrage Apache : sudo systemctl reload apache2
  • Vérification dans le navigateur à l'adresse http://localhost/
