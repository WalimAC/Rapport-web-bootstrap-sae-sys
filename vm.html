Virtualisation

Description de la création et configuration de la machine virtuelle Linux.

Configuration de la VM

Paramètres de la machine virtuelle :

  • Nom : SAE_S1.03_VM_Gr106_BinomeG2
  • OS invité : Ubuntu 22.04 LTS
  • RAM : 4GB
  • Disque : 40GB dynamique (VDI)
  • Réseau : NAT

Étapes détaillées :

  1. Télécharger et installer VirtualBox depuis le site officiel (virtualbox.org)
  2. Créer une nouvelle VM avec les paramètres ci-dessus
  3. Monter l'ISO Ubuntu 22.04 LTS pour l'installation
  4. Installer l'OS étape par étape : sélection langue français, clavier français, partitionnement automatique

Utilisateurs et groupes

Configuration des comptes utilisateurs et groupes :

  • Administrateur : admin_Gr106_BinomeG2 (droits sudo)
  • Utilisateur 1 : BinomeG2_1
  • Utilisateur 2 : BinomeG2_2
  • Groupe partagé : BinomeG2

Commandes utilisées :

# Création de l'administrateur avec droits sudo
sudo adduser admin_Gr106_BinomeG2
sudo usermod -aG sudo admin_Gr106_BinomeG2

# Création des deux utilisateurs
sudo adduser BinomeG2_1
sudo adduser BinomeG2_2

# Création du groupe partagé
sudo groupadd BinomeG2
sudo usermod -aG BinomeG2 BinomeG2_1
sudo usermod -aG BinomeG2 BinomeG2_2

Personnalisation du prompt pour chaque utilisateur :

echo 'PS1="\u@\h:\w\$ "' >> ~/.bashrc
source ~/.bashrc
